Temperature and light visualization

Here in this we have two visualizations

1. Temperature Bar Graph
In this Bar Graph we have temperature values of recent 2 hours data pushed in the datasparkfun.com stream url. Temperature is recorded at each location using DS18B20 temperature sensor. This is Dynamic Graph which will be changing whenever there is an entry in the Class stream for IoT

2. Light Bar Graph
In this Bar Graph we have light values recorded by user's raspberry pi. This light is calculated using photosensor. It has recent 2 hours data pushed by all users into the datasparkfun.com stream url. This is Dynamic Graph which will be changing whenever there is an entry in the Class stream for IoT
