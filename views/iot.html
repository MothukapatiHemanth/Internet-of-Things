About Project

The internet of Things is like a connection of a group of physical objects,sensors or things such as raspberry pi, wifi dongle, keyboard, mouse and display etc , software as tightvncserver, sensors as photosensor & DS18B20 temperature sensor, and connectivity is made among the objects where the communication can happen between the objects and the aggregator where the data is collected/placed i.e like data in a database or data in a website. This data is analaysed and visualized. Other connected devices are absed on the required infrastructure.

Generally in this project I have used temperature sensor DS18B20,MPC3008 for analog to digital conversion and photoresistor for measuring the temperature in celsius and luminous intensity. I used sensors with combination of raspberry pi to measure the temperature and light values. We used breadboard, cobler, temperature sensor, phototransistor, 10K ohms resistor and few connection wires for creating a circuit. Here we pushed data to the data sparkfun website by using the http get request in python programming. Data sparkfun acted as aggregator which collected the temperature and light readings of all their particular locations using sensors.

I have created a visualization by performing an analysis on the data available in sparkfun. That visulazitaion can beviewed by using the pythonanywhere link. Here I have used Twilio for getting a text message which provides the temperature of particular location where I placed my sensors. I am adding mail functionality where the temperature and light intensity of particular location will be send to my mobile for every 6 hours.

Coming to future enhancements I am planning to get an message whenevere there is a immediate change in temperature and luminous intensity value goes very low. I have chosen this project i.e mainly temperature measurement because ohio is one of the places which experiences all type of weather as too much hot, snow, raining and spring. By this project we can get the temperature information of a particular location in ohio.
